How to build a tutorial with Steps.js

The HTML guided tutorial framework

In this tutorial you will create a fully working tutorial with Tutorial JS

What you'll need

  • Basic understanding of HTML and CSS.
  • Something to teach.

The HTML

           
                <div class="stepsjs"
                     data-title="Build a tutorial with Steps.js"
                     data-selected="0"
                     data-toolbar="true"
                     data-arrows="true">
                </div>
